Group Wallets

Pool money together.
Spend with a vote.

Enable a shared wallet in any Telegram or WhatsApp group. Members contribute cUSD, and every spend goes through a democratic poll. No single person controls the money.

  • ✓
    Democratic PollsEvery purchase needs group approval. Configurable threshold (default 70%). Native Telegram and WhatsApp polls.
  • ✓
    Contribute & TrackMembers contribute from their personal wallet. Everyone gets a private DM with deduction details.
  • ✓
    Admin ControlsAdmins can bypass polls for urgent purchases, configure poll expiry, export group wallet keys, and withdraw funds.
  • ✓
    Group ReportsGenerate PDF or Excel statements of all group transactions. Full transparency for every member.
Real examples

Just say what you need

Toppa handles complex requests in plain language. Here's what real users say.

"Send my brother 500 naira airtime, pay mom's DStv bill, and buy me a $25 Steam gift card"
3 operations Toppa parses this into three parallel tool calls — detects MTN Nigeria, finds the DStv biller, searches Steam gift cards — and creates all three order confirmations in a single turn.
"Set up weekly airtime for my wife every Friday — 1000 naira to her MTN line"
Recurring Creates a scheduled task that runs every Friday. The heartbeat engine checks every 15 minutes and auto-sends when it's due. Retry with backoff if delivery fails.
"Everyone pitch in 5 cUSD for the office WiFi bill this month"
Group wallet Posted in a group chat. Members contribute from their personal wallets. When someone says "pay the bill," a poll goes out. Once 70% approve, Toppa pays the ISP directly.
Voice note: "Buy a 2GB data bundle for this number 0802..."
Voice Transcribed via Deepgram in under a second. Toppa detects the operator from the number, finds the best 2GB plan, and creates the order confirmation. Works in English, French, Yoruba, Swahili.
"Swap all my CELO and USDC to cUSD"
Uniswap V3 Routes through the Uniswap Trading API for the best rate. Falls back to direct SwapRouter02 if the API rejects the quote. Shows a before/after breakdown of all balances.
"Give me a report of everything I've spent this month"
Reports Generates a PDF or Excel file with every transaction — airtime, data, bills, gift cards, swaps, group contributions. Delivered as a document right in the chat.
Protocol

Powered by x402 on Celo

Toppa uses the x402 HTTP payment protocol on the Celo blockchain. No API keys, no subscriptions — just pay per request with cUSD.

x402 HTTP Payment Protocol

The x402 protocol turns HTTP 402 into a machine-readable payment flow. Any agent or app can call Toppa's API — payment happens inline with a single cUSD transaction on Celo. No sign-up, no API keys.

1
Request Agent calls the API
2
402 Response Server returns price
3

4
Delivered Result returned

Celo Blockchain

Fast finality (~5s), sub-cent gas fees, mobile-first. cUSD stablecoin means no price volatility.
