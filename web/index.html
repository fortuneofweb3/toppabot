Built for agents & developers

Model Context Protocol

13 tools available via Streamable HTTP. Connect any MCP-compatible client directly.

POST api.toppa.cc/mcp
A2A

Agent-to-Agent Protocol

Google's A2A standard. Send tasks to Toppa from other agents via JSON-RPC.

POST api.toppa.cc/a2a

Pay-Per-Use API

REST endpoints with x402 payment headers. Pay in cUSD, no API keys needed.
