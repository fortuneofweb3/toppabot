Built for Agents & Developers

Integrate Toppa into your own agents, apps, and workflows.

MCP

Model Context Protocol

12 tools available via Streamable HTTP. Connect any MCP-compatible client directly.

POST api.toppa.cc/mcp
A2A

Agent-to-Agent Protocol

Google's A2A standard. Send tasks to Toppa from other agents via JSON-RPC.

POST api.toppa.cc/a2a
x402

Pay-Per-Use API

Standard REST endpoints with x402 payment headers. Pay in cUSD, no API keys needed.

POST api.toppa.cc/send-airtime
ERC-8004

On-Chain Identity

Registered agent on Celo with on-chain reputation tracking and verifiable identity.

GET api.toppa.cc/identity
Discovery

Free Discovery APIs

Browse operators, billers, gift cards, and promotions by country. No auth required.

GET api.toppa.cc/operators/NG

Auto-Discovery
